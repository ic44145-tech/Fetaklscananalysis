This folder contains the architecture in the respective .py files and the trained model weights for each tasks as .h5 files.

These are as follows:
1. bvc_xception: Brain view classification model weights
2. fl_unet: Femur length estimation segmentation model weights
3. hc_unet: Brain biometry measurement segmentation model weights
4. uspc_xception: Ultrasound plane classification model weights

.py files:
1. unet_model: Segmentation model architecture
2. xception_model: Classification model architecture
